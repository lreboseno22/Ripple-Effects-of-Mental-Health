# The Ripple Effects of Mental Illness

Small presentation-style site built with plain HTML, CSS and JavaScript that explores youth mental health indicators in NYC.

## Overview

- Purpose: A seven-slide deck highlighting youth mental health statistics, community impacts, and resources. 
- Tech: Vanilla HTML5, CSS3, and JavaScript. Charts are rendered with Chart.js. No build step required.

Engineer: Liam Reboseno

---

## Project Structure

- `index.html` — main presentation and slides
- `styles.css` — all styles for the deck and slides
- `script.js` — interactive behavior, slide navigation, background cycling, and chart initialization
- `resources/` — CSV data used for slide 4
	- `Youth Feeling Sad or Hopeless by Race-Ethnicity.csv`
	- `Youth Feeling Sad or Hopeless by Gender.csv`
	- `Youth Feeling Sad or Hopeless by Borough.csv`
	- `Youth Feeling Sad or Hopeless by Sexual Orientation.csv`
